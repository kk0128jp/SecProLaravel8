# SecureLaravel9

## 概要

「[IPA 安全なウェブサイトの作り方　改訂第7版](https://www.ipa.go.jp/security/vuln/websecurity/ug65p900000196e2-att/000017316.pdf)」を元にWebアプリケーションの脆弱性をいくつか取り上げた。

その脆弱性に対してLaravelでの対策例を実装した。

※対策の実装はあくまで一例です。

## 脆弱性一覧

下記の脆弱性と対策を実装しました。

1. SQLインジェクション
2. OSコマンドインジェクション
3. ディレクトリトラバーサル
4. クロスサイト・スクリプティング
5. CSRF(クロスサイト・リクエスト・フォージェリ)
